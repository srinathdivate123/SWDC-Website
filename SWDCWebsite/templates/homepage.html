Tech Stack

DJango

DJango's robustness and well-structured architecture made it a suitable choice for this website handling many users

Python

Python is used to add other backend functionality like multithreading, generating PDFs, sheets, sending mails, etc

Linux

This website is hosted on a Linux environment as Django deployment tools (like Gunicorn, Nginx, and Docker) integrate seamlessly with Linux

Other Python Libraries

GeoPy, OpenPyxl, Pillow, PyPDF2, PyYAML, QrCode, x-HTML-2-PDF are used to handle backend tasks

SQL Database

A SQL database is used to store data of all the users. This project has 13 database tables

JavaScript

JavaScript provides the optimal user experience and also handles some part of the communication with server using requests

What's the purpose of this website?

First Year students of VIT Pune have to do a Social Service Course in their freshman year.This website is used for their registrations, attendance, report filling and report verification.

Registration happen division-wise, each division of students is eligible for a particular set of activities and students have to choose among the activities.

The activities happen in various areas of Pune. Attendance of students is marked by coordinators and requires the students to be within 2km of the activity location, pre-determined by a Google Maps link.

Once the activities are completed, students have to fill a report of it. The report filling page does not allow pasting answers into it, nor does it allow exiting full screen or changing tabs. If students install extensions to counter these, the extensions are tracked & blocked

Coordinators verify the reports filled by students, and on approval, students are mailed a activity completion certificate and responses of their report

What is the future of this website?

Every student who takes admission into VIT Pune has to do the Social Services Course in their Freshman Year. So this website is doomed to grow in terms of the number of users and logins.

As long as PythonAnywhere keeps hosting websites for free, this website would keep running forever!
