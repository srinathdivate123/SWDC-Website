Instructions for registration of Volunteers

  • Make sure you have set the correct Academic Year and Semester in the Current-Data table. This is very crucial.
  • You may enable the desired activity for registration, and set its current count as 0 and the max count as desired.
  • If an activity is marked true for only males, then females will not be able to register. Same applies vice-versa.
